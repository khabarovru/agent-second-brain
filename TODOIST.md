# TODOIST.md — Синхронизация Obsidian ↔ Todoist

## Принцип

**Всё в Todoist, дубликат в Obsidian.** Todoist — источник истины для задач. Obsidian — резервная копия и расширенный контекст.

## Синхронизация

### Todoist → Obsidian (автоматически)

Скрипт `todoist-sync-closed.py` запускается по cron в 21:00 и проверяет закрытые задачи. Если задача закрыта в Todoist — отмечается галочкой в Obsidian.

**Формат записи в Obsidian:**
```markdown
- [x] [[TaskName]] — выполнено в Todoist
```

### Obsidian → Todoist (автоматически)

Когда создаёшь задачу в Obsidian с тегом `sync-todoist`, она автоматически попадает в Todoist через агента.

**Формат:**
```yaml
---
created: YYYY-MM-DD HH:MM
type: task
tags: [sync-todoist]
priority: 2
todoist_sync: true
---

# Название задачи

Контекст и детали...
```

### Ручная синхронизация (галочка в Obsidian)

Если поставить галочку `☐ → ☑` в Obsidian:
1. Агент автоматически отмечает задачу выполненной в Todoist
2. Записывает в метаданные: `completed_obsidian: YYYY-MM-DD`

## Структура задач в Obsidian

```
50 Задачи/Active/
├── YYYY-MM-DD Название задачи.md    # Активные задачи
└── Completed/
    └── YYYY-MM-DD Название задачи.md  # Выполненные
```

---

## Формат frontmatter для задач

```yaml
---
created: YYYY-MM-DD HH:MM
type: task
tags: [sync-todoist, todoist]
source: [vk-text|telegram-text|...]
priority: 1-4
todoist_id: "XXXXX"          # ID из Todoist (после создания)
due: YYYY-MM-DD
related: [["alias или link"]]
completed_obsidian: YYYY-MM-DD
completed_todoist: YYYY-MM-DD
---

# Название задачи

## Контекст
- [[Павловник]] — кому отдать
- [[Дельта]] — локация
- [[2026-04-20]] — дедлайн

## Чек-лист
- [ ] Пункт 1
- [ ] Пункт 2
```

## Aliases (линки в задачах)

Используй `[[alias]]` синтаксис для связывания с сущностями:

| Тип | Пример | Что связываем |
|-----|--------|--------------|
| **Люди** | `[[Павловник]]`, `[[Полонникова Татьяна]]` | 70 Люди/* |
| **Места** | `[[Дельта]]`, `[[Офис 9046]]` | 70 Контексты/* |
| **Даты** | `[[2026-04-20]]`, `[[deadline]]` | 20 Ежедневник/Daily Notes/* |
| **Проекты** | `[[лифсервис Альфа]]` | 60 Проекты/* |
| **Заметки** | `[[Документы]]` | 10 Заметки/* |

## Команды

| Действие | Как сделать |
|----------|-------------|
| Создать задачу | Запиши в `00 Входящие/Tasks/` или скажи агенту |
| Посмотреть задачи | `📊 Статус` → секция Todoist |
| Отметить выполненным | Поставь галочку в Obsidian или в Todoist |
| Удалить задачу | Только в Todoist (Obsidian обновится при синхронизации) |

---

## Cron jobs (только Todoist!)

**Больше НЕ создаю напоминания через cron jobs!** Все напоминания — через Todoist.

| Cron | Зачем |
|------|-------|
| Syncthing conflict resolution | Каждый час |
| Todoist → Obsidian sync | 21:00 daily |
