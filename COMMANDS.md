# COMMANDS.md - Обработка команд (читать ПЕРВЫМ при команде!)

## 🔄 /new или /reset

**Когда:** при старте новой сессии (контекст сбросился)

**Что делать:**
1. Прочитать `99 Сервис/Контекст Saver.md` — там процедура
2. Извлечь ключевую информацию из предыдущей переписки
3. Сохранить контекст в `memory/Контексты/YYYY-MM-DD.md`
4. Обновить `40 Объекты/устройства.md` если появились новые устройства/IP
5. Обновить MEMORY.md если появились долгосрочные данные

**Искать:** IP-адреса, токены (частично!), устройства, решения, ожидающее

---

## 📊 /obsidian_status или 📊 Статус

**Вызвать скрипт:**
```bash
exec agent/scripts/vault-status.sh
```

Скрипт вернёт готовую статистику — просто отправь её владельцу как есть.

---

## /process

**Вызвать скрипт:**
```bash
exec agent/scripts/vault-process.sh
```

Скрипт раскладывает записи дня по папкам и возвращает отчёт — просто отправь его владельцу.

---

## /do <запрос>

**Что делать:**
Выполнить произвольный запрос в контексте vault.

**Примеры:**
- `/do найди все незакрытые задачи` → искать в vault/50 Задачи/Active/, показать список
- `/do покажи идеи за неделю` → читать vault/00 Входящие/Ideas/, фильтровать по дате
- `/do перенеси просроченные задачи на понедельник` → найти + обновить due_date

---

## /weekly

**Что делать:**
1. `read vault/20 Ежедневник/Daily Notes/YYYY-MM-DD.md` (последние 7 дней)
2. Создать сводку:
   - Главные темы
   - Завершённые задачи
   - Новые идеи
   - Ключевые инсайты
3. `write vault/reports/weekly-YYYY-WW.md`
4. Ответить кратким дайджестом

---

## /help

**Что делать:**
Показать список команд (вывести содержимое этого файла в кратком виде)
