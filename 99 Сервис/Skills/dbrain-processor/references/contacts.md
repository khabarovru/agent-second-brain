# Contacts

## Search Protocol
When looking for a person or contact:
1. Search vault CRM files by name
2. Check daily files for recent mentions
3. Search by company association

## Contact Sources
- CRM cards in `business/crm/`
- Project records in `projects/`
- Daily logs for recent interactions
