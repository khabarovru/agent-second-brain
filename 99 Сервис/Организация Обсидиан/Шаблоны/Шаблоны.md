# Шаблоны
## Цели и задачи

- [[🎯 Шаблон Цели]] — шаблон для создания заметок с долгосрочными целями.
- [[📂 Шаблон Проекта]] — шаблон для создания заметки, прикрепленной к папке проекта (используй вставку шаблона Alt+Shift+H). Полная инструкция в .
- [[Шаблон - ✅ Задача]] — шаблон для создания задачи, для создания жми *Alt+T* → напиши текст задачи.
- [[Шаблон видео]] — шаблон для создания задачи типа "видео" (подумал, может пригодится).
- [[📑 Шаблон Допник]] — шаблон для создания необходимых документов для задачи/проекта (сценарий, пост, список, документ и тд). Путь заметки выбираешь при создании.
- [[📑 Шаблон Допник]] — также этот шаблон используется для заметки, у которой путь по умолчанию: ==Нефильтрованое/Допники== — используй как допник, разница только в том, что тут не нужно выбирать путь заметки.

## Ежедневные заметки

- [[📝 Шаблон блога]] — заметка для записи всех процессов дня и постановки задач (тасков) — создаётся через календарик.
- [[🗓 Шаблон ежедневной заметки]] — заметка с дашбордами для мониторинга задач, идей и собственных комментариев, а также для подведения итогов дня — создаётся через [[Блог|блог]].
- [[📅 Шаблон еженедельной заметки]] — заметка для подведения итогов недели.
- [[📔 Шаблон Дневник]] — заметка для написание своих мыслей в начале/в течение/в конце дня (произвольная форма, — пиши как хочешь и что хочешь). Рекомендации по ведению в [[Дневник|дневнику]].

## Тематические заметки

- [[🔹 Базовый шаблон]] — заметка на свободную тему, для которой путь по умолчанию: ==00 Входящие/Базовые заметки==
- [[Пустой шаблон]] — заметка на свободную тему, путь которой выбираешь при создании.
- Также, когда ты кликаешь по несуществующим ссылкам, то у тебя создаётся заметка, к которой применяется [[🔹 Базовый шаблон]] и данная заметка падает в папку ==00 Входящие/Без фильтра==

## Телеграм заметки

- [[📒 Шаблон Блокнота Телеграм]] — заметка в блокнот
- [[🔑 Шаблон Ключи]] — заметка для записи паролей и ключей, через телеграм (используй шифратор)
- [[📚 Шаблон Книги для Чтения]] — заметка для пополнения списка книг, которые ты хотел бы прочитать

> [!INFO]+  Информация
Телеграм заметки создаются автоматически, с помощью . Но для этого сперва нужно его настроить.

## Другие

- [[EPUB новая книга]] — заметка, которую ты можешь превратить в книгу, просто добавь pdf или epub файл в папку ==50 Библиотека/Формат книги== → затем создай через  новую заметку в разделе <font color="#ffc000">Книги</font> → в созданной заметке в поле <span style="background:#ff4d4f">annotation-target</span> выбери свою книгу → перейди в режим чтения, нажав <span style="background:#40a9ff">Alt+Shift+R</span> → profit
- [[🎯 Шаблон Цели]] — шаблон для заметки с описанием и для отслеживания какой-то одной цели
- [[📌 Шаблон Темы]] — шаблон для создания новой темы
- [[📂 Шаблон Проекта]] — шаблон для создания нового проекта
- [[📈 Шаблон привычки]] — шаблон для создания заметки по новой привычке
