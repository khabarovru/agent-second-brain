# Настройка автоматических задач для Second Brain

Эти кроны нужно создать **из самого агента** (отправить команды в @khabarovru_obsidian_bot).

---

## 1. Daily Processing (ежедневная обработка)

**Когда:** Каждый день в 21:00 МСК  
**Что делает:** Обрабатывает все записи дня и раскладывает по папкам

**Команда для создания:**
```
Создай крон на 21:00 каждый день:

Выполни /process:
1. Прочитай vault/20 Ежедневник/Daily Notes/YYYY-MM-DD.md (сегодняшний)
2. Классифицируй все записи
3. Перенеси задачи в vault/50 Задачи/Active/
4. Перенеси идеи в vault/00 Входящие/Ideas/
5. Перенеси знания в vault/90 Карты знаний/Knowledge Base/
6. Отправь краткий отчёт

Используй модель sonnet-4-5, таймаут 10 минут
```

---

## 2. Weekly Digest (недельный дайджест)

**Когда:** Каждое воскресенье в 20:00 МСК  
**Что делает:** Создаёт сводку за неделю

**Команда для создания:**
```
Создай крон на воскресенье 20:00:

Выполни /weekly:
1. Прочитай все daily notes за последние 7 дней
2. Создай сводку:
   - Главные темы недели
   - Завершённые задачи
   - Новые идеи
   - Ключевые инсайты
3. Сохрани в vault/reports/weekly-YYYY-WW.md
4. Отправь дайджест

Используй модель sonnet-4-5, таймаут 15 минут
```

---

## 3. Vault Health Check (проверка связности)

**Когда:** Каждую пятницу в 18:00 МСК  
**Что делает:** Проверяет "здоровье" vault

**Команда для создания:**
```
Создай крон на пятницу 18:00:

Проверь vault:
1. Найди orphan notes (без ссылок)
2. Найди broken links
3. Предложи связи между заметками
4. Проверь теги (нет ли дубликатов)
5. Отправь отчёт с рекомендациями

Используй модель sonnet-4-5, таймаут 10 минут
```

---

## 4. Git Auto-Commit (уже создан)

✅ Уже настроен: Daily Vault Auto-Upload (22:00 UTC)  
Проверяет git status и коммитит изменения.

---

## Как создать:

1. Открой Telegram → @khabarovru_obsidian_bot
2. Отправь команду из раздела выше (например, для Daily Processing)
3. Агент создаст крон и подтвердит
4. Проверь: отправь "📊 статус" — должны появиться активные кроны

---

## Проверка активных кронов:

Отправь агенту:
```
Покажи все активные кроны для second-brain
```

Должно быть:
- Daily Processing (21:00 МСК)
- Weekly Digest (воскресенье 20:00 МСК)
- Vault Health Check (пятница 18:00 МСК)
- Git Auto-Commit (22:00 UTC) ✅
