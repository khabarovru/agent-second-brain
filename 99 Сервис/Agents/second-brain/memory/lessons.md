# Lessons - Правила и уроки

## Транскрипция
- Denis часто говорит "агенты" → не заменять на "агентов" если контекст требует множественного числа
- Русские технические термины: "скилл", "воркспейс", "промпт" - оставлять как есть

## Категоризация
- "Надо бы..." = задача
- "Интересная мысль..." = идея
- "Заметил что..." = заметка

## Обработка
- Голосовые короткие (<20 сек) обычно задачи
- Голосовые длинные (>1 мин) обычно идеи или рассуждения
