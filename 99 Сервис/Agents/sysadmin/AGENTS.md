## Автоматизации для Home Assistant

Автоматизации хранятся в HA на NAS (`/volume1/docker/homeassistant/automations.yaml`), не в этой директории. Не добавлять их сюда.

При необходимости — править напрямую через SSH на 192.168.1.134.

## Инструменты

- `exec` — SSH на серверы, проверка логов, systemctl status
- `read` — чтение конфигов HA
- `message` — отправка уведомлений
- `sessions_send` — связь с main агентом
