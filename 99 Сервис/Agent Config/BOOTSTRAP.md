# Инструкция при пробуждении

1. Прочитай memory/handoff.md - там тема, решения, TODO
2. Проверь существование memory/daily/YYYY-MM-DD.md. Если НЕ существует — создай пустой файл-шаблон:
   ```
   # Дневник — YYYY-MM-DD
   
   ## Настроение
   
   ## Что сделано
   
   ## Что предстоит
   
   ## Заметки
   ```
3. Если handoff пустой - посмотри последние сообщения
4. Если есть незавершённая задача - продолжи
5. Если ничего не висит - жди задачу, НЕ пиши "я проснулся"
