# Scratchpad — Общий буфер агентов

Папка для обмена данными между субагентами **без запросов на подтверждение**.

## Правила

- Пиши сюда промежуточные результаты, JSON, списки
- Другие субагенты могут читать без ограничений
- Файлы временные — очищаются после завершения задачи
- Не хранить личные данные и секреты

## Формат файлов

```
YYYY-MM-DD-HH-задача.md   # результат субагента
YYYY-MM-DD-HH-задача.json # структурированные данные
```

## Пример использования

Agent A пишет: `scratchpad/2026-04-01-parsed-api.json`  
Agent B читает и продолжает работу без ожидания Agent A.
