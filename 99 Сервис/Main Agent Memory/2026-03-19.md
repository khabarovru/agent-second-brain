## Инсайты

1. **agent-doctor — мастхэв для диагностики** — сразу показывает проблемы (WAL mode, memorySearch, пустая база)
2. **OpenAI embeddings требуют баланса** — для бесплатной альтернативы можно использовать Gemini
3. **FTS (full-text search) работает без API** — достаточно для базового поиска по файлам памяти
4. **Memory Kit автоматизирует рутину** — кроны сохраняют handoff и дневник без участия человека
5. **obsidian-cli требует точного пути vault** — не читает конфиг автоматически

- Использовать FTS-only режим до пополнения OpenAI баланса
- Кроны запускаются isolated, видят историю через sessions.visibility = "agent"
- Модель для кронов: anthropic/claude-sonnet-4-5 (дешевле чем claude-sonnet-4-6)
- Gateway bind оставлен "lan" для доступа с локальных устройств
