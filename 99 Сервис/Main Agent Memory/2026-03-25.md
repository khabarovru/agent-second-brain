**Ключевые решения:**
1. Один gateway с двумя ботами (вместо двух gateway)
2. Маршрутизация через bindings по accountId
3. Команды выполняются через скрипты (надёжнее чем прямые tool-вызовы)
4. Tool session_status отключен для агента (tools.deny)
5. Доступ только для Denis (258505)

**Workflow:**
Голос/текст → транскрипция → классификация (задача/идея/заметка/знание) → сохранение в vault/memory/YYYY-MM-DD.md → команда /process раскладывает по папкам

**Команды:**
- `/obsidian_status` - статистика vault
- `/process` - обработка записей
- `/do <запрос>` - произвольное действие
- `/weekly` - недельный дайджест
