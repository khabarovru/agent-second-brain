# Паттерны самообучения

Формат: ситуация → что пошло не так → паттерн → счётчик повторений
Если счётчик ≥ 3 → создать правило в lessons-learned.md

---

## 2026-03-24

**Паттерн: Проверка auth-profiles перед выводами**
- Ситуация: Сказал что API-ключ мёртвый, агент работает "по инерции"
- Ошибка: Проверил только openclaw.json, не заглянул в auth-profiles.json
- Правило: Всегда проверять auth-profiles.json — там реальный активный токен
- Повторений: 1

**Паттерн: WAL mode слетает**
- Ситуация: WAL сбрасывается на delete после перезапуска gateway
- Ошибка: OpenClaw не устанавливает WAL при открытии базы
- Правило: Проверять WAL при каждом цикле watchdog
- Повторений: 3 → ✅ правило добавлено, ensure-wal встроен в watchdog

**Паттерн: Не показывать токены в чат**
- Ситуация: Показал фрагменты ключей при проверке конфига
- Ошибка: Даже частичный вывод ключа — риск
- Правило: Только "есть/нет" и тип. Никогда фрагменты
- Повторений: 1
