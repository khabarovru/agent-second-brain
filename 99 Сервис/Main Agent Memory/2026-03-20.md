# 2026-03-20 - Дневник дня

## 09:00-10:30 - Настройка d-brain бота с Tor

**Задача:** Запустить Telegram-бота d-brain через Tor SOCKS5 прокси для анонимного подключения.

**Что сделали:**
- Установили зависимости: `aiohttp-socks`, `python-socks`
- Проверили работу Tor-сервиса (systemd)
- Несколько итераций правок `d_brain/bot/main.py`:
  - Первая попытка: monkey-patch `AiohttpSession._session` — не сработало
  - Вторая попытка: создали кастомный класс `TorSession` с переопределённым `create_session()`
- Столкнулись с rate-limit 429 от Anthropic API при отладке
- Процесс запустился, логи показывают `Using Tor SOCKS5 proxy`, но `Cannot connect to host` при попытке подключения к Telegram

**Решения:**
- Переход на наследование `AiohttpSession` для правильной интеграции с aiogram
- Использование `nohup` для фонового запуска с логированием

**Инсайты:**
- aiogram требует правильной инициализации session _до_ создания `ClientSession`
- Tor на порту 9050 работает, но коннект через SOCKS5 не устанавливается — возможно, проблема в настройках Tor или aiogram
- Нужно тестировать прокси отдельно (curl через SOCKS5) перед интеграцией

**Статус:** Бот запущен, но не подключается к Telegram через Tor. Требуется дальнейшая отладка.
