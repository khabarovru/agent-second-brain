## 🔧 Dataview Views

### project-progress
Прогресс-бар задач проекта. Вызов: `dv.view('project-progress')`
- Показывает % выполнения
- Список задач с чекбоксами
- Цвет меняется по прогрессу (красный → фиолетовый → зелёный)
