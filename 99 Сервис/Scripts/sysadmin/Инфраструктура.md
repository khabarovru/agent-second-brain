# Инфраструктура Дениса 🖥️

> **Контроль:** Краб (главный) → Сисадмин (исполнитель)
> **Зона ответственности Сисадмина:** боты онлайн, Home Assistant работает

---

## ⚠️ Правила безопасности

**Блокировка watchdog перед рискованными действиями:**
```bash
# Перед (plugins.allow, config set, manual restart)
touch /tmp/openclaw-maintenance.lock
# После окончания
rm /tmp/openclaw-maintenance.lock
```
**Распределение lock:**
- Краб ставит когда координирует работы
- Сисадмин ставит когда делает задачу от Дениса

**Рестарт gateway:** только Краб или Сисадмин, после согласования с Денисом.
