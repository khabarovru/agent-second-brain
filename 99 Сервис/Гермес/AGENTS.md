# AGENTS.md — Команды для Гермеса

## Формат входящих сообщений

Каждое сообщение — это мысль для обработки. Команды начинаются с `/`.

---

## Команды

### /Статус — Статистика vault
Показать статистику за сегодня: задачи, идеи, заметки, знания.
**Работает только здесь (Hermes/Telegram).**
```bash
~/.hermes/scripts/vault-status.sh
```

### /процесс — Обработка входящих
Обработка входящих записей: показывает необработанные файлы в корне 00 Входящие и сводку за день.
**Работает только здесь (Hermes/Telegram).**
```bash
~/.hermes/scripts/vault-process.sh
```

### /дейли — Дневной дайджест
Генерация дайджеста: входящие за вчера, превью дневника, новые заметки, напоминание Todoist.
**Работает только здесь (Hermes/Telegram).**
```bash
~/.hermes/scripts/daily-digest.sh
```

### /коуч — Коуч-режим
Анализ целей и прогресса, проверка задач из Todoist, формирование рекомендаций.
**Работает только здесь (Hermes/Telegram).**
```bash
~/.hermes/scripts/coach-mode.sh
```

### /неделя — Недельный дайджест
Агрегирует данные за неделю: новые записи, прогресс по целям, активность.
```bash
~/.hermes/scripts/weekly-digest.sh
```

### /стата [дни] — Статистика взаимодействий
```bash
python3 ~/.hermes/scripts/session-stats.py 7
```

### process-incoming.sh — Обработка входящих
Создаёт задачи из 00 Входящие в Todoist.
```bash
~/.hermes/scripts/process-incoming.sh --dry-run  # тест
~/.hermes/scripts/process-incoming.sh            # рабочий
```

### `/tasks` — Активные задачи
Показать открытые задачи из Todoist.

### `/today` — Что на сегодня
Задачи на сегодня из Todoist + ежедневная заметка.

### `/inbox` — Входящие
Последние записи из 00 Входящие/.

---

## Обработка мысли (без команды)

**Шаг 1:** Сохранить сырую мысль в `00 Входящие/[Type]/YYYY-MM-DD Название.md`

**Шаг 2:** Осмыслить:
1. Определить тип (задача/идея/заметка/знание)
2. Найти связи в vault
3. Дополнить контекстом
4. Обогатить тегами

**Шаг 3:** Если задача → создать в Todoist

**Шаг 4:** Уведомить кратко

---

## Голосовые

При получении голосового:
1. Скачать аудио (VK: webm/ogg, Telegram: ogg/opus)
2. Транскрибировать через Deepgram (nova-2, ru)
3. Обработать как текстовую мысль
4. Удалить аудио после обработки
