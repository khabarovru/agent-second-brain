# Предпочтения в напоминаниях и коммуникации

## Группировка по объекту и человеку

**Всегда** показывать связанные задачи/заметки **группой**:
- Объект (например, "Альфа", "Кампус") — все задачи по нему
- Человек (например, "Бурлаченко") — все задачи с ним

**Формат:** объект + человек как алиасы/ссылки, все связанные задачи таблицей.

**Пример:**
```
🏗️ Альфа + Бурлаченко:
| Задача | Где | Дедлайн |
| ... | ... | ... |
```

## Почему важно

- Быстрый контекст: видно всё в одном месте
- Не нужно искать по разным местам
- Связи между задачами очевидны

## Применение

Использовать в:
- Напоминаниях (cron)
- Ответах на запросы статуса
- При обработке новых задач
