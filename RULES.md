# RULES.md - Правила Second Brain

## 🚨 Главные правила

### 1. НИКОГДА не удалять файлы без согласования

**НИКОГДА не удалять файлы без согласования с владельцем.**

Даже если:
- Файлы "бесполезные", "без линков", "не нужны"
- Скрипт показал что файл "осиротел"
- Кажется что файл "мусор"
- Редактирование стёрло wikilink'и

**Сначала спросить.** Если нельзя спросить — не удалять.

**Вывод из инцидента 2026-04-22:**
1. Сначала кривые правки стёрли wikilink'и → файл стал "без линков"
2. Потом скрипт удалил "осиротевшие" файлы
3. Результат: потеря данных

**Правило:** Если файл существует в vault — он нужен. Удалить можно только после подтверждения владельца.

### 2. Syncthing — резервная копия, не git

**Git = version control для пустой структуры папок. Реальные данные — в Syncthing.**
